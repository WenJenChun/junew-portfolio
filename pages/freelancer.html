台灣人在英國生活情報

這是一個以 React 為框架的資訊網站，目標使用者為在英國生活的台灣人。靈感來自於過去在英國打工期間，深刻體會在異國生活的種種挑戰，想找資訊卻碰上網路社群資源、訊息分散等不易搜尋問題，因此萌生建立這個資訊平台的念頭。

目前網站仍在開發中，內容將使用 json-server-auth 及 Render 完成 API 串接。
